# scrum-pocker Backend

### REQUIREMENTS:

- Nodejs
- Express
- MongoDB

Steps to run the Backend side:

    1. Install the dependencies

    ```
    $ npm install
    ```

    2. Run the server

    ```
    $ npm run start
    ```

Then the project is going to start, but before connect the database.

NOTE:
